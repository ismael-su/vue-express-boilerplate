This is a tutorial showcasing how to build a vuejs app and serve it using expressjs

# install express 
```shell
npm install express --save
```

# Build  production 
```shell
cd src
npm build
```
this will generate a folder named dist
take a look at the index.js file


## Start app
```
node index
```
